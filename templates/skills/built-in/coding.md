# Coding Skill

Write, modify, and review code with automated validation.

## Protocol

1. Understand the requirements and constraints
2. Read existing code in the target files
3. Plan the implementation approach
4. Write or modify code following project conventions
5. Run tests to validate changes
6. Create a PR with clear description

## Guidelines

- Follow existing code style and conventions
- Write tests for new functionality
- Keep changes focused and minimal
- Run the test suite before committing
- Use meaningful variable and function names
- Handle errors appropriately
